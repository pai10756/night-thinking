## 二、跨平台爆款的 7 大共通元素

### 1. 前 3 秒決勝負
- 65% 看完前 3 秒的人會看到 10 秒
- 70%+ 的「前 3 秒留存率」是爆款門檻
- **三合一 Hook**：視覺衝擊（動態畫面）+ 文字 Hook（3-7 字）+ 語音 Hook（5-10 字）

### 2. 真實感 > 完美感
- 過度製作的影片反而表現差
- 「今天的爆款影片看起來是人做的，不是廣告公司做的」
- UGC 風格（手持、自然光、直視鏡頭）>專業攝影棚感

### 3. 一支影片只講一件事
- 每支影片一個核心訊息
- 觀眾記住的不是「你講了很多」而是「你講得很清楚」

### 4. 可行動 > 純知識
- 「告訴我該怎麼做」比「這是什麼原理」表現好 3 倍
- 步驟化（❶❷❸）比段落式文字更容易被記住

### 5. 情緒觸發是分享引擎
| 情緒 | 觸發方式 | 分享動機 |
|------|---------|---------|
| 驚訝 | 反直覺事實、顛覆認知 | 「你知道嗎？我也嚇到」|
| 關懷 | 家人健康、長輩安全 | 「傳給爸媽看」|
| 實用 | 可立即執行的建議 | 「收藏起來」|
| 認同 | 「這在說我！」的共鳴 | 「@朋友 這就是你」|

### 6. 循環結構（Looping）
- 結尾接回開頭，觀眾不自覺重播
- 重播 = 演算法判定為高品質 = 更多推薦
- 適用於：結果揭曉型、「你注意到了嗎？」型

### 7. 社交貨幣價值
- 觀眾分享影片是為了「讓自己看起來有見識」
- 反直覺健康知識 = 最強社交貨幣（「原來不是這樣！」）

## 三、10 種高效短影片結構模板

### 模板 1：3 個快速技巧（30 秒）
```
Hook（3s）→ 技巧 1（7s）→ 技巧 2（7s）→ 技巧 3（7s）→ CTA（6s）
```
適合：健康建議、飲食提醒、運動技巧

### 模板 2：Before/After 對比
```
Hook + Before 畫面（5s）→ 解釋原因（10s）→ After 畫面（10s）→ CTA（5s）
```
適合：姿勢矯正、飲食改變、睡眠改善

### 模板 3：POV 故事（30-45 秒）
```
「POV：你是...」設定（3s）→ 場景演示（15s）→ 轉折/揭示（10s）→ 教訓（5s）
```
適合：日常健康場景、就醫經驗

### 模板 4：「你做錯了」糾錯
```
Hook「你一直都做錯了」（3s）→ 錯誤示範（5s）→ 正確做法（15s）→ CTA（5s）
```
適合：運動姿勢、用藥方式、飲食習慣

### 模板 5：微教學（15-30 秒）
```
Hook（3s）→ 1-2 步示範（15s）→ 結果展示（5s）→ CTA（5s）
```
適合：穴位按摩、伸展動作、呼吸法

### 模板 6：「我試了 X 天」實驗
```
Hook + 實驗設定（5s）→ 過程快速剪輯（15s）→ 結果數據（5s）→ 心得（5s）
```
適合：飲食實驗、運動挑戰、睡眠方法

### 模板 7：倒數排名
```
Hook「Top 3...」（3s）→ #3（8s）→ #2（8s）→ #1（8s）→ CTA（5s）
```
適合：最佳食物、最有效運動、最常見錯誤

### 模板 8：回覆留言
```
展示留言問題（3s）→ 1-3 點回答（20s）→ 邀請更多提問（5s）
```
適合：建立社群互動、回覆健康疑問

### 模板 9：反轉驚喜
```
熟悉的設定（5s）→ 突然反轉（3s）→ 解釋真相（15s）→ CTA（5s）
```
適合：健康迷思破除、食品真相

### 模板 10：專家訪問
```
問題引入（3s）→ 專家回答（20s）→ 行動建議（5s）→ CTA（5s）
```
適合：醫學新知、營養建議、運動安全

## 四、中老年觀眾的特殊考量

### 他們在意什麼
- 家人健康（為了下一代/上一代）
- 自身慢性病管理
- 不想成為家人的負擔
- 維持獨立生活能力

### 內容偏好
- 說人話，不要醫學術語
- 具體可執行（「明天就能開始做」）
- 有權威背書（期刊、醫生、機構）
- 正面框架（「做這個更好」而非「不做就死」）

### 觀看習慣
- 字幕大且清晰（視力考量）
- 語速適中不要太快
- 重複關鍵訊息 2-3 次
- 金色/白色字幕在深色背景上最清楚
