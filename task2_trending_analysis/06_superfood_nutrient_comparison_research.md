## 一、為什麼「食物營養素比較」特別容易爆量？

### 1.1 核心爆量心理機制

| 心理觸發 | 說明 | 範例 Hook |
|----------|------|-----------|
| **認知衝擊** | 觀眾發現自己長期以來的認知是錯的 → 「我一直都吃錯了？」 | 「補鈣別再喝骨頭湯了！這個便宜菜含鈣量是牛奶的 3 倍」 |
| **排行榜效應** | 人天生對「排名」有強迫式好奇心，想看到底誰第一 | 「含鐵量最高的食物 TOP 5，第一名不是菠菜！」 |
| **數字對比衝擊** | 具體數字比模糊描述更有說服力，差距越大越震撼 | 「100g 芝麻含鈣 975mg，牛奶只有 104mg」 |
| **實用價值** | 「學到了可以馬上用」的即時獲得感 | 「想補鐵？這 3 樣菜市場就有，便宜又有效」 |
| **社交貨幣** | 分享者扮演「知識傳遞者」角色，展現關心家人 | 「趕快傳給媽媽看！她煮的那道菜其實超營養」 |
| **迷思破除快感** | 推翻「常識」帶來的優越感和傳播慾 | 「紅棗補血？醫學上完全沒有根據」 |

### 1.2 最容易爆量的內容格式

#### 格式 A：「XX vs XX」對決型
```
畫面分左右兩邊，同時展示兩種食物的營養素數據
→ 觀眾天然想知道「誰贏」
→ 適合：牛奶 vs 豆漿、白米 vs 糙米、雞蛋 vs 豆腐
```

#### 格式 B：「TOP 排行榜」倒數揭曉型
```
從第 5 名開始倒數到第 1 名
→ 排行榜效應 + 懸念感驅動完播率
→ 適合：含鈣食物 TOP 5、抗氧化食物排行榜
```

#### 格式 C：「你以為 vs 事實」顛覆認知型
```
先展示「大眾認知」→ 再用數據打臉
→ 反差越大，分享率越高
→ 適合：菠菜含鐵量的迷思、骨頭湯補鈣的真相
```

#### 格式 D：「一張圖看懂」資訊圖表型
```
將多種食物的某營養素含量做成視覺化長條圖/圓餅圖
→ 收藏率極高（小紅書種草效應強）
→ 適合：維生素 C 食物排行、蛋白質含量比較
```

#### 格式 E：「這個竟然比那個還營養」驚奇型
```
用一個便宜/常見食物 PK 一個「公認健康」的貴食物
→ 天然省錢社交貨幣 + 認知衝擊
→ 適合：「地瓜葉的鈣含量竟然比牛奶還高？」
```

### 1.3 爆量關鍵要素清單

- [ ] **數字要具體**：「含鈣 975mg」比「含鈣很高」強 10 倍
- [ ] **對比要極端**：差距越大越震撼（3 倍、5 倍、10 倍）
- [ ] **來源要標注**：畫面角落標注「資料來源：USDA」增加可信度
- [ ] **食材要常見**：觀眾家裡廚房就有的食材 >> 異國超級食物
- [ ] **結論要顛覆**：如果結果跟大眾認知一樣，就沒有傳播力
- [ ] **可行動建議**：不只告訴「是什麼」，還要說「怎麼吃」
- [ ] **3 秒鉤子**：開頭直接拋出最驚人的數字或結論

## 二、「超級食物」主題的特殊優勢與陷阱

### 2.1 為什麼適合【長輩廚房】

| 優勢 | 說明 |
|------|------|
| **長輩最關心飲食** | 飲食是 50+ 歲族群第一關注話題，分享率極高 |
| **天然社交貨幣** | 「這個菜竟然比 XX 還營養」是最好的飯桌話題 |
| **破除迷思空間大** | 長輩群體中流傳大量未經證實的飲食迷思（紅棗補血、骨頭湯補鈣等） |
| **省錢角度加持** | 用便宜食材替代昂貴保健品的內容，在長輩圈轉發率最高 |
| **季節性內容充足** | 可搭配節氣、季節食材做常態化排程 |

### 2.2 必須注意的陷阱

| 陷阱 | 說明 | 對策 |
|------|------|------|
| **「超級食物」本身是行銷術語** | 歐盟 2007 年已禁止使用此標籤，無公認醫學定義 | 避免使用「超級食物」一詞，改用「高營養密度食物」或直接說營養素含量 |
| **TikTok 營養內容僅 2.1% 準確** | 大量創作者引用錯誤數據，觀眾會質疑 | 每個數據都標注出處（USDA、PubMed），建立專業信任 |
| **脫離脈絡的數字會誤導** | 例如芝麻含鈣高但一次不會吃 100g | 必須換算成「實際一餐的攝取量」而非 100g 單位比較 |
| **生物利用率被忽略** | 植物性鈣的吸收率遠低於乳製品鈣 | 提及營養素時加上「吸收率」或「搭配 XX 一起吃效果更好」 |
| **過度神化單一食物** | 暗示某食物能「治病」會被平台降權或被事實查核 | 框架為「均衡飲食的一部分」，不做療效宣稱 |
| **哈佛研究已打臉 ORAC 數據** | USDA 曾發布抗氧化排行榜後又撤回，因為抗氧化劑功能比原先認知複雜 | 避免使用過時的 ORAC 排名，引用最新系統性回顧 |

### 4.3 差異化策略：為什麼「長輩廚房」的營養比較能脫穎而出

目前 TikTok 營養內容的問題是 **僅 2.1% 準確**，這反而是機會：

1. **標注資料來源** → 在一片不標來源的內容中，標注「USDA」「PubMed」立即建立信任差異化
2. **換算實際份量** → 不只比 100g，而是比「一碗飯」「一杯牛奶」「一把菠菜」的實際攝取量
3. **提及吸收率** → 「芝麻含鈣高但吸收率只有 20%，牛奶鈣吸收率 30%」→ 這種深一層的知識讓觀眾覺得「這個頻道是認真的」
4. **溫暖語氣** → 不恐嚇（「不吃就會死」），而是正面框架（「換這個吃更好」）
5. **長輩視角** → 「媽媽煮了一輩子，這個小調整讓全家更健康」觸發家人分享

## 五、製作工作流程建議

### 5.1 資料查證流程

```
1. 選定主題（如「補鈣食物排行」）
2. USDA FoodData Central 查詢各食物的鈣含量數據
3. 台灣食品營養成分資料庫交叉驗證本地食材數據
4. PubMed 搜尋「[nutrient] bioavailability systematic review」確認吸收率
5. Harvard Nutrition Source 查看該營養素的通俗解讀
6. MyGoPen 查詢相關說法是否已被列為謠言
7. 換算成實際餐食份量（非僅 100g 比較）
8. 撰寫腳本，每個數據標注出處
```

### 5.2 USDA FoodData Central 實用搜尋技巧

- 直接搜尋食物英文名（如 "spinach, raw"）
- 選擇 **SR Legacy** 或 **Foundation Foods** 數據類型（品質最高）
- 可按單一營養素排序所有食物 → 直接得到排行榜數據
- API 可批量下載數據（適合大量比較）：https://fdc.nal.usda.gov/api-guide.html
