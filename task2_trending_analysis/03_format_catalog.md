# 影片形式目錄（含 Seedance 2.0 模板）

本目錄定義 8 種視覺形式，供 50 個健康主題分配使用。
每種形式均包含 Seedance 全域規格、場景圖 prompt 範本、適合主題類型。

---

## 形式 A：演播室主播（Studio Anchor）

**數量分配**：8/50
**適合主題**：權威性高的醫學發現、重大研究結果、需要嚴肅感的主題

### Seedance 全域規格
```
电影级短片，影视制作摘要，9:16竖屏。
全域规格：neutral-cool色温，高反差，冷白主光，深蓝石墨灰背景，白色边缘灯条，干净玻璃桌面。
```

写实摄影风格，9:16竖屏。

## 跨形式共通規則

1. **所有場景圖必須無人**（Seedance 人臉 filter 限制）
2. **所有口播/旁白指定「台湾口音普通话」**
3. **每句 6-12 字，句句斷開**
4. **每個 shot 只做一件事**
5. **Sound 描述簡短（5-10 字）**
6. **Music 一律「不抢语音」**
7. **結尾禁止語：「禁止：任何文字、字幕、LOGO或水印。」**
