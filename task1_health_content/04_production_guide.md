# 多形式生產指南

> 本指南說明如何將 50 個健康主題的 Seedance 劇本，搭配現有 `health-digest-factory` 管線進行批量生產。

---

## 一、生產流程概覽

```
01. 從 02_seedance_prompts.json 取得劇本
02. 在即夢平台（jimeng.jianying.com）生成場景圖（無人場景）
03. 在即夢平台生成 Part1 影片（15s）
04. 在即夢平台生成 Part2 影片（15s）
05. 下載兩段影片到 health-digest-factory 根目錄
06. 修改 assemble_seedance.py 的參數（標題、字幕）
07. 執行 python scripts/assemble_seedance.py
08. 產出 720x1280 ~33s 影片
09. 從 03_youtube_metadata.json 取得標題與說明
10. 上傳到 YouTube Shorts
```

## 二、各形式的生產差異

### 形式 A：演播室主播
- **場景圖**：1 張演播室（固定可重用）+ 2 張主題相關場景
- **主播一致性**：先跑 Part2 滿意後，Part1 加「同上一部短片亚洲女性」
- **delogo 參數**：標準 `x=640:y=1195:w=70:h=75`
- **BGM**：沉穩低調，現有 default.mp3 適用

### 形式 B：戶外自然
- **場景圖**：3 張不同戶外場景（公園/步道/花園/湖畔，依主題選）
- **人物描述**：prompt 中描述「60岁亚洲长者」而非上傳人臉
- **特別注意**：戶外場景色調偏暖，assemble 時不需額外調色
- **BGM**：需更換為溫暖輕柔鋼琴曲

### 形式 C：街頭訪問
- **場景圖**：1 張街頭/市場 + 1-2 張主題場景
- **人物描述**：「30岁亚洲女记者，手持麦克风」
- **鏡頭風格**：prompt 指定「手持轻微晃动」
- **BGM**：輕快活潑，需另選曲

### 形式 D：醫療專家
- **場景圖**：1 張診所辦公室（可重用）+ 1-2 張醫療相關場景
- **人物描述**：「45岁亚洲男医生，白袍」
- **信任感要素**：背景書架、聽診器等道具在場景圖中呈現
- **BGM**：溫和弦樂，營造信任感

### 形式 E：廚房居家
- **場景圖**：1 張廚房（可重用）+ 1-2 張食材特寫/餐桌場景
- **人物描述**：「55岁亚洲女性，站在料理台前」
- **特別注意**：食材特寫場景要寫明具體食物名稱
- **BGM**：溫暖歡快，居家氛圍

### 形式 F：運動示範
- **場景圖**：1 張公園/客廳運動場景 + 1 張動作特寫場景
- **人物描述**：「60岁亚洲长者，穿着舒适运动服」
- **鏡頭**：需要全身景到局部特寫的切換
- **BGM**：輕快正能量

### 形式 G：對比轉變
- **場景圖**：1 張冷色調「問題」場景 + 1 張暖色調「改善」場景
- **色調控制**：prompt 中明確指定「冷蓝色调」和「暖金色调」
- **轉場**：Part1→Part2 的色調轉換是這個形式的核心
- **BGM**：從低沉到積極的變化曲

### 形式 H：紀錄片空鏡
- **場景圖**：2-3 張壯觀自然景觀（日出/森林/田野）
- **人物**：可以沒有人物，純旁白 + 空鏡
- **鏡頭**：航拍、超廣角推進
- **BGM**：史詩感弦樂

## 三、場景圖可重用清單

以下場景圖可在多個主題間重複使用，減少生成次數：

| 場景 | 可重用主題 | 數量 |
|------|-----------|------|
| 演播室（深藍 LED 牆） | 所有 A 形式 | 8 次 |
| 公園步道（金色晨光） | B 形式中的運動/散步類 | 5-6 次 |
| 廚房料理台 | 所有 E 形式 | 6 次 |
| 診所辦公室 | 所有 D 形式 | 6 次 |
| 台北街頭 | 所有 C 形式 | 4-5 次 |
| 昏暗臥室（冷色） | G 形式的 Before | 2-3 次 |
| 明亮臥室（暖色） | G 形式的 After | 2-3 次 |

**預計需生成的獨立場景圖**：約 30-40 張（重用後）

## 五、字幕統一規範

依 SEEDANCE_WORKFLOW.md：

| 元素 | 設定 |
|------|------|
| 字型 | Microsoft JhengHei Bold |
| 主播播報字幕 | 52px, 白色 |
| 旁白字幕 | 48px, 白色 |
| 步驟字幕 | 44px, 白色 |
| 重點句高亮 | 天藍 `0x00BFFF` 或金黃 `0xFFD700` |
| 描邊 | Outline=4, Shadow=1, 黑色 |
| MarginV | 220（避開 Shorts UI） |
| 浮水印 | 右上角 22px 半透明白「時時靜好」全程顯示 |

## 六、批量生產建議排程

每週生產 5-7 支影片，按以下順序安排：

| 週 | 主題編號 | 形式分佈 | 備註 |
|---|---------|---------|------|
| 1 | 1-7 | A×2, C×2, G×1, B×1, E×1 | 混合開局 |
| 2 | 8-14 | C×1, D×1, A×1, E×2, B×1, G×1 | 加入廚房/專家 |
| 3 | 15-21 | B×2, C×1, A×1, F×2, D×1 | 引入運動示範 |
| 4 | 22-28 | B×2, F×1, A×1, H×1, C×1, G×1 | 引入紀錄片 |
| 5 | 29-35 | C×2, H×1, A×1, G×1, D×2 | 認知+慢性病 |
| 6 | 36-42 | D×2, E×1, F×1, B×1, H×1, B×1 | 慢性病+心理 |
| 7 | 43-50 | C×2, B×2, E×1, A×2, D×1 | 收尾多樣化 |

```python
SUBTITLES = [
    # 從 subtitles 陣列取

```

可考慮未來改為讀取 JSON 自動帶入，減少手動修改。

## 八、品質檢查清單

每支影片上傳前確認：

- [ ] 影片長度 30-35 秒
- [ ] 片頭 Hook 文字清晰可讀
- [ ] 字幕與語音同步（±0.3s 以內）
- [ ] 字幕不被 Shorts UI 遮擋（MarginV≥220）
- [ ] 右上角浮水印「時時靜好」可見
- [ ] 即夢星形浮水印已去除
- [ ] 主播/人物外觀 Part1 和 Part2 基本一致
- [ ] 語音為台灣口音，語速適中
- [ ] BGM 不搶語音
- [ ] YouTube 標題 < 60 字元
- [ ] 說明含研究來源、步驟、hashtags
