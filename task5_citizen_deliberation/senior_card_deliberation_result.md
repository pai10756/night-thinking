# 敬老卡使用優化 — 公民審議模擬紀錄
日期：2026-03-16 00:58

## 政策背景
## 台灣敬老卡現行政策摘要

### 基本制度
- 對象：65歲以上長者（原住民55歲以上）
- 形式：結合悠遊卡/一卡通，每月自動儲值點數（1點=1元）

### 各縣市補助差異
| 縣市 | 月補助點數 | 主要使用範圍 |
|------|-----------|-------------|
| 台北市 | 480點 | 公車、捷運、計程車(每趟上限65元)、YouBike、運動中心 |
| 新北市 | 480點 | 公車、捷運、計程車、YouBike |
| 桃園市 | 800點 | 公車、計程車(每趟上限100元)、運動中心 |
| 台中市 | 1,000點 | 公車免費、計程車(每趟上限85元)、運動中心 |
| 新竹市 | 800點 | 公車、運動中心(含游泳池)、藥局 |
| 高雄市 | 1,200點/年 | 公車、輕軌、渡輪、計程車 |

### 現行問題
1. 點數限當月使用，不可累積至次月
2. 使用範圍主要限於交通，生活應用有限
3. 各縣市標準不一，跨縣市使用受限
4. 部分長者不熟悉數位操作，使用率偏低
5. 補助金額是否足夠、是否應依需求差異化
6. 計程車補助上限低，對行動不便者不足

### 本次審議目標
優化敬老卡的使用，使其更好地服務長者的多元需求，同時確保財政可持續性。

**【王大姐・66歲社區志工】**：主持人、各位夥伴大家好。聽到陳阿嬤和林伯伯的建議，我真的很有感觸，他們說的都是長輩們最直接的心聲。點數累積、提高計程車補助上限、擴大使用範圍，這些都是非常實際且重要的方向。

特別是陳阿嬤提到「看醫生、買菜沒煩惱」的心願，以及林伯伯說「政府不要只想著控制預算」，這些話都讓我想到社區裡那些平常不太出門的長輩。他們不是不願意參與，而是真的有困難。

我服務的長輩中，有些是因為身體退化，有些是因為獨居少與人互動，還有一些是資訊比較不靈通，對於這些新的卡片功能或數位服務感到卻步。所以，我的建議會更著重在「如何主動把資源送到長輩身邊」和「提升社會參與的誘因」。

首先，我贊成點數應該可以「彈性累積」至少三個月，讓長輩更有時間運用。但更重要的是，我們能不能針對「長期不出門」或「行動不便」的長輩，提供「居家服務點數」？這些點數可以讓他們僱請居家服務員協助購物、打掃，甚至陪同外出散步，這不僅能減輕長輩的負擔，也能讓他們感受到社會的溫暖，活絡居家經濟。

再來，為了縮小數位落差，我建議可以舉辦「社區敬老卡使用教學班」，由年輕志工或社區發展協會來協助長輩學習如何使用敬老卡搭計程車App、查詢特約商店，甚至教他們如何在運動中心預約課程。甚至可以考慮讓社區發展協會或關懷據點代為申請或預約服務，讓長輩不需親自操作。

最後是**智慧科技的付費模式**。我認為應採「**政府補助基礎功能，使用者付費加值，並對弱勢長者全額補助**」。這能平衡財政壓力與服務需求，讓有能力的家庭能為長輩加值，同時確保弱勢長者不會因經濟因素而失去保障。但前提是，所有加值服務都必須經過嚴格的資安與隱私審查。

**【黃醫師・家醫科醫師（健康促進視角）】**：主持人、各位與會者，大家好。很高興能看到大家對於敬老卡優化有這麼多的共識，這對我們推動長者健康真的很有幫助。

對於「延長點數使用期限」、「擴大使用範圍至健康促進活動」以及「預防勝於治療的預算概念」，我當然是**百分之百支持**。這些都是我長年來在基層看診的深刻體會。長輩多出門、多活動，不只是身體會更健康，更重要的是心理狀態也會更穩定，失智症發生的機率也會降低，這對減少未來的醫療和長照負擔，絕對是最好的投資。

至於「計程車補助金額」，我也強烈建議**支持提高**，至少應統一提高到單趟100元。我看到太多長輩因為膝蓋退化、心血管疾病，甚至跌倒受傷後，無法搭乘大眾運輸，但卻因為計程車補助不足，捨不得叫車來回診，導致病情延誤。這不是我們樂見的。對於身障或經醫師評估行動不便的長輩，更應該給予更高的補助上限，甚至可以考慮實報實銷但設有總額的方案，讓他們能安心就醫。這筆錢是為了確保他們的就醫可近性，是非常必要的投資。

關於「智慧科技導入」，我是**有條件支持**。陳阿嬤的疑慮我非常理解，操作複雜確實會讓長輩卻步。但像林伯伯提到的線上查詢、預約功能，以及張先生擔心的遠端關懷，這些都是可以簡化操作並兼顧隱私的。我建議政府應優先推動簡單易用的基礎功能，並透過社區據點的志工提供教學，甚至可以結合我們的家醫診所，在長輩回診時順便教導。至於進階功能如一鍵求助或GPS追蹤，在充分告知並徵得長輩及家屬同意的前提下，可以作為加值服務，但必須確保資料加密和隱私保護，這是底線。

「預算來源與財政永續性」，我同意這是一項「投資」，不是單純支出。我認為中央應訂定統一的最低補助標準，並透過跨部會協調，將部分健保或長照預算，更有效地引導至前端的預防保健。同時，地方政府也應積極向中央爭取專案補助，並建立政策效益評估機制，證明這些投資能有效降低醫療與長照支出，形成良性循環。

最後是「智慧科技的付費模式」，我支持「**政府補助基礎功能，使用者付費加值，並對弱勢長者全額補助**」。這樣既能確保所有長輩都能享有基本保障，又能讓有需求的家庭為長輩提供更多元的安全加值服務。重要的是，政府必須對這些加值服務的資安和隱私保護進行嚴格審查。

## 敬老卡使用優化政策建議報告

**日期：** 2026年3月16日
**審議主題：** 敬老卡使用優化

### 1. 審議摘要

本次公民審議針對台灣敬老卡現行政策的諸多問題進行了深入討論，包括點數當月歸零、使用範圍狹隘、各縣市標準不一、數位落差及補助金額不足等。參與者涵蓋長者、退休公務員、社區志工、照顧者、市議員及家醫科醫師，多元視角提供了豐富建議。

審議結果明確指出敬老卡應從單純的交通補助轉型為「健康促進」與「社會參與」的整合性工具。主要共識包括延長點數使用期限、擴大至健康促進活動。分歧點則聚焦於計程車補助幅度、智慧科技導入的複雜性與隱私疑慮，以及最關鍵的預算來源與財政永續性。報告將這些共識與分歧轉化為具體政策建議，並評估其可行性、預算影響及實施優先序，旨在讓敬老卡更能貼近長者需求，同時確保財政可持續性。

### 2. 主要共識點

本次審議中，所有或多數參與者已達成以下共識：

1.  **點數使用期限彈性化：** 一致同意將敬老卡點數的當月使用期限延長，建議至少延長至三個月，甚至可考慮半年，以避免點數浪費，提升長者使用彈性。
2.  **擴大使用範圍至健康促進活動：** 普遍支持將敬老卡點數使用範圍擴大，優先納入公立運動中心、公立游泳池、社區關懷據點共餐費、社區大學課程及其他有益於身心健康的藝文活動等。
3.  **「預防勝於治療」的預算概念：** 各方認同將敬老卡視為前端「健康投資」，透過鼓勵長者積極參與健康活動，從長遠來看可有效降低未來的醫療與長照支出，提升整體社會效益。
4.  **重視數位落差與簡易操作：** 承認部分長者不熟悉數位科技，新功能設計應以「簡單、直覺」為原則，並透過社區據點、志工或專業人員提供教學與協助。
5.  **照顧弱勢長者：** 應特別關懷獨居、失智或行動不便的長者，探討提供居家服務點數，或由社區據點協助代辦服務等，確保他們也能受益。
6.  **中央應制定最低保障標準：** 各縣市福利標準不一造成不公平感，建議中央政府應制定全國性的敬老卡最低補助標準與使用指引，確保基礎福利一致性。
7.  **政策效益評估機制：** 應建立一套有效的政策效果評估機制，定期檢討敬老卡的效益，確保每一分錢都花得值得，並作為後續預算編列的參考。

### 3. 主要分歧點

本次審議仍存在以下主要分歧點，需進一步協調或採取多元方案：

1.  **計程車補助金額與範圍：**
    *   **支持提高：** 長者與照顧者普遍認為現行單趟65元補助過低，建議提高至80-100元，並對行動不便者提供更高額補助。
    *   **財政考量：** 蔡議員認為全面提高補助對市庫負擔沉重，建議有條件支持，如提高至80元，並針對身障或經評估行動不便者提供更高上限。
2.  **智慧科技導入的程度與隱私：**
    *   **支持導入：** 照顧者強調GPS追蹤、一鍵求助、健康數據上傳可提升長者安全與照顧者安心。
    *   **疑慮：** 長者對操作複雜性與隱私權有疑慮，擔心被監控。
    *   **平衡：** 建議從簡單功能推動，並在充分告知、徵得同意下提供進階加值服務。
3.  **預算來源與財政永續性：**
    *   **重分配：** 黃醫師與林伯伯建議從健保或長照預算中提撥部分經費，轉向前端預防。
    *   **新增財源：** 蔡議員強調需考慮是否有其他稅收來源，或更有效率的行政成本控制，並積極向中央爭取專案補助。
    *   **公私協力：** 張先生建議政府與科技業者合作，或引入民間資源。
4.  **智慧科技的付費模式：**
    *   **使用者付費加值：** 張先生建議政府提供平台，讓家屬可自費加值，享受更多進階安全功能。
    *   **福利公平性：** 蔡議員擔心使用者付費會造成「福利階級化」，強調政府應補助基礎功能，並對弱勢長者全額補助。

### 4. 具體政策建議

基於上述共識與分歧，提出以下分階段政策建議：

#### 短期（未來3個月內）

1.  **延長點數使用期限至三個月：**
    *   **建議內容：** 立即修正敬老卡點數「當月歸零」規定，延長使用期限至至少三個月。系統應能自動累積與扣除，無需長者額外操作。
    *   **具體做法：** 函文各縣市政府，要求修正點數管理辦法，並協調票證公司修改系統設定。
    *   **預算影響：** 無直接增額支出，僅為點數管理方式調整。
    *   **受益群體：** 所有敬老卡使用者，特別是每月點數使用不穩定的長者。
2.  **計程車補助上限統一提高至80元：**
    *   **建議內容：** 參考物價與桃園市標準，將現行計程車每趟補助上限統一提高至80元。
    *   **具體做法：** 中央發函建議各縣市政府調整補助標準，並可提供地方政府部分經費支持。
    *   **預算影響：** 根據各縣市使用量，預計每月增加約20%~30%的計程車補助支出。以台北市為例，若每月計程車使用點數為總點數20% (96點)，每趟平均補助從65元提高到80元，每次增加15元，則96/65*15 = 約22元/月/人。全國每月約30萬人次使用計程車，總體增加約450萬元/月。
    *   **受益群體：** 所有有搭乘計程車需求的長者，特別是行動不便者。
3.  **啟動「健康促進」使用範圍試點計畫：**
    *   **建議內容：** 優先選擇部分縣市，將敬老卡使用範圍擴大至公立運動中心、公立游泳池、社區關懷據點共餐費，並提供專案補助。
    *   **具體做法：** 由衛福部或教育部體育署與地方政府合作，選定5-8個具備條件的縣市作為試點，提供額外經費誘因。
    *   **預算影響：** 試點計畫預計每年需投入約5,000萬至1億元中央補助款，用於設備升級、人員培訓及點數折抵。
    *   **受益群體：** 試點縣市內有意願參與健康促進活動的長者。

#### 中期（未來1年內）

1.  **推動全國性敬老卡使用標準最低保障：**
    *   **建議內容：** 中央政府與各縣市協商，制定全國統一的敬老卡最低補助標準與使用指引，確保基礎福利一致性，不足部分由中央專案補助。
    *   **具體做法：** 召開跨部會會議（衛福部、內政部、交通部、財政部），研擬全國統一規範，並規劃中央補助款分配機制。
    *   **預算影響：** 中央政府需編列每年約5億至10億元專案補助款，以彌補財政較弱縣市的福利差距。
    *   **受益群體：** 全國所有長者，特別是財政較弱縣市的長者，可享有更公平的福利。
2.  **建立敬老卡「數位落差友善計畫」：**
    *   **建議內容：** 在全國社區據點、關懷據點或圖書館，開設敬老卡使用教學班，由志工或專人協助長者學習卡片操作、查詢點數、預約服務等。
    *   **具體做法：** 衛福部與地方政府合作，培訓志工，提供教學教材，並設置專屬服務櫃檯或諮詢電話。
    *   **預算影響：** 每年約需投入5,000萬至1億元，用於志工培訓、教材開發、據點設備更新及宣傳。
    *   **受益群體：** 不熟悉數位操作的長者，特別是獨居或缺乏家人協助的長者。
3.  **導入敬老卡「基礎智慧安全功能」：**
    *   **建議內容：** 發展操作簡單、隱私無虞的基礎智慧功能，例如線上查詢點數餘額、搭乘大眾運輸記錄查詢，並可選擇性綁定緊急聯絡人。
    *   **具體做法：** 與票證公司及科技業者合作，開發敬老卡專屬App或網頁介面，初期由政府全額補助。
    *   **預算影響：** 系統開發與維護每年約需1億至2億元，包含資安防護與後台維護。
    *   **受益群體：** 所有敬老卡使用者及關心長者安全的照顧者。

#### 長期（未來3年內）

1.  **深化「預防保健」與「健康回饋」機制：**
    *   **建議內容：** 將敬老卡點數擴大至公立醫療院所的掛號費、部分負擔費用（特定項目），或作為完成年度健康檢查、施打疫苗的獎勵點數。
    *   **具體做法：** 衛福部與健保署協商，評估可行性，並規劃點數回饋機制。可考慮將部分健保預算轉移至此，以達成前端預防效益。
    *   **預算影響：** 初期每年約需3億至5億元，用於點數補助與獎勵。長期透過減少醫療與長照支出，預計可節省更多。
    *   **受益群體：** 所有長者，鼓勵其積極參與預防保健，降低慢性病風險。
2.  **推動「敬老卡加值服務」與「弱勢保障」雙軌制：**
    *   **建議內容：** 針對更進階的智慧安全功能（如GPS追蹤、一鍵求助、跌倒偵測），建立「使用者付費加值」模式，同時政府對經濟弱勢長者提供全額補助。
    *   **具體做法：** 公開招標科技業者開發平台，提供多樣化加值服務方案。政府設定弱勢長者資格門檻，並編列專款補助。
    *   **預算影響：** 平台開發與維護約每年1億元。弱勢長者補助視需求量而定，預計每年約2億至5億元。使用者付費可分攤部分成本。
    *   **受益群體：** 有意願使用進階智慧安全功能的長者及照顧者，同時確保弱勢長者不會因經濟能力受限而失去安全保障。
3.  **建立「敬老卡點數共享機制」：**
    *   **建議內容：** 鼓勵長者將每月未使用完畢的點數（或部分點數）捐贈至專屬基金，集中運用於社區弱勢長者居家服務、送餐服務或緊急醫療補助。
    *   **具體做法：** 衛福部與地方政府合作，設立點數捐贈平台，並制定點數分配與使用辦法。
    *   **預算影響：** 主要為既有點數的活化與重新分配，行政管理費用較低，預計每年約500萬至1,000萬元。
    *   **受益群體：** 獨居、行動不便或經濟弱勢的長者，以及所有願意分享點數的長者。

*   **地方政府自籌：** 地方政府可依其財政能力，在中央最低標準基礎上進行加碼，或爭取地方型公益彩券盈餘等。
*   **公私協力：** 鼓勵企業與民間團體參與，共同推動敬老卡服務，例如提供加值服務選項。
*   **使用者付費加值：** 針對進階智慧功能，引入使用者付費模式，減輕政府財政壓力，並對弱勢長者提供全額補助。

### 6. 實施優先序建議

基於本次審議的共識程度、預算可行性與預期效益，建議以下實施優先序：

1.  **第一優先（短期實施，3個月內）：**
    *   **延長點數使用期限至三個月：** 共識度高，無直接增額支出，技術可行性高，長者最有感。
    *   **計程車補助上限統一提高至80元：** 解決長者交通不便的迫切需求，預算增幅可控。
    *   **啟動「健康促進」使用範圍試點計畫：** 驗證「預防勝於治療」概念，為全國推廣奠定基礎，預算有中央支持。
    *   **優先理由：** 這些措施能快速回應長者當前最普遍的痛點，提升使用便利性與福利有感度，且預算增幅相對可控。

2.  **第二優先（中期實施，1年內）：**
    *   **推動全國性敬老卡使用標準最低保障：** 解決各縣市福利不均的根本問題，需中央與地方協商，但具備高度公平性與社會效益。
    *   **建立敬老卡「數位落差友善計畫」：** 解決長者數位障礙，確保福利涵蓋所有長者，提升政策有效性。
    *   **導入敬老卡「基礎智慧安全功能」：** 在長者可接受的範圍內，提升其安全與照顧者安心感，為未來智慧化奠定基礎。
    *   **優先理由：** 這些措施著重於制度面的完善、公平性的提升以及基礎科技的導入，是長期發展的基石，需要較長的規劃與協調時間。

3.  **第三優先（長期實施，3年內）：**
    *   **深化「預防保健」與「健康回饋」機制：** 實現「預防勝於治療」的願景，效益巨大但執行複雜，需跨部會深度合作。
    *   **推動「敬老卡加值服務」與「弱勢保障」雙軌制：** 提供更個人化服務，同時確保福利公平，需完善的平台與管理機制。
    *   **建立「敬老卡點數共享機制」：** 活化資源，照顧弱勢，需社會共識與信任機制建立。
    *   **優先理由：** 這些是更具前瞻性和複雜性的方案，涉及更大規模的資源重分配、技術整合和社會參與，需要更多時間進行規劃、測試與推廣。
